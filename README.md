# Design MD

Biblioteca open-source de templates `DESIGN.md` em português para UX, UI, Product Design e ferramentas de IA.

Projeto criado por CamaraUX.

🔗 https://camaraux.com.br

---

# O que é Design MD?

`DESIGN.md` é um formato estruturado utilizado para descrever interfaces, sistemas de design, componentes, fluxos e regras de experiência para ferramentas de IA, geração de interfaces e workflows de produto.

Esses arquivos podem ser utilizados em ferramentas como:

- Claude
- Cursor
- v0
- Lovable
- Bolt
- Figma AI
- ChatGPT
- Windsurf
- Ferramentas agentic AI

---

Disponibilizar templates prontos de `DESIGN.md` em português para acelerar:
